Modules task Object Scheduler

Object : Scheduler

Represents a ballerina task Scheduler, which can be used to run jobs periodically, using the given configurations.

Constructor

Initializes a task:Scheduler object. This may panic if the initialization causes any error due to a configuration error.

__init

(TimerConfiguration | AppointmentConfiguration configuration)

  • configuration TimerConfiguration | AppointmentConfiguration

  • The task:TimerConfiguration or task:AppointmentConfiguration record to define the task:Sceduler behavior

Methods

attach

Attaches the provided service to the task.

detach

Detaches the provided service from the task.

start

Starts running the task.

stop

Stops the task.

pause

Pauses the task.

resume

Resumes a paused task.

isStarted

Checks whether the task listener is started or not.

attach 

        function attach(service serviceToAttach, any... attachments) returns SchedulerError?

Attaches the provided service to the task.

Parameters

  • serviceToAttach service

    • Ballerina service object, which needs to be attached to the task

  • attachments any...

    • Set of optional parameters, which need to be passed inside the resources

        function start() returns SchedulerError?

isStarted 

        function isStarted() returns boolean

Checks whether the task listener is started or not.

Return Type

(boolean)

true if the Scheduler is already started or else false if the Scheduler is not started yet or stopped calling the Scheduler.stop() function
