contains 

        function contains(string key) returns boolean

Checks whether the given key is in the configuration registry.

 boolean configAvailable = config:contains("host");

Parameters

  • key string

    • The configuration key to be looked-up

Return Type

(boolean)

Returns true if the key is present; if not returs false

getAsArray 

        function getAsArray(string key) returns null

Retrieves the specified configuration value as an array.

 int[]|error ports = config:getAsArray("ports").cloneWithType(int[]);

Parameters

  • key string

    • The key of the configuration to be retrieved

Return Type

(null)

Configuration value mapped with the given key. If there is no mapping, an empty array will be returned

getAsBoolean 

        function getAsBoolean(string key, boolean defaultValue) returns boolean

Retrieves the specified configuration value as a boolean.

 boolean cachingEnabled = config:getAsBoolean("http.caching_enabled");

Parameters

  • key string

    • The key of the configuration to be retrieved

  • defaultValue boolean (default false)

    • The default value to be used in case there is no mapping for the provided key

Return Type

(boolean)

Configuration value mapped with the given key

setConfig 

        function setConfig(string key, string | int | float | boolean value)
