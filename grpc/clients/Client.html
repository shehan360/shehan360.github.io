Clients : Client

The gRPC client endpoint provides the capability for initiating contact with a remote gRPC service. The API it provides includes functions to send request/error messages.

Constructor

Gets invoked to initialize the endpoint. During initialization, the configurations provided through the config record are used for the endpoint initialization.

__init

(string url, ClientConfiguration? config)

  • url string

  • The server URL

  • config ClientConfiguration? <ballerina/grpc:0.7.0:ClientConfiguration?> ()
    • The grpc:ClientConfiguration of the endpoint
