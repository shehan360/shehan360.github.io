Modules auth Object InboundBasicAuthProvider

Object : InboundBasicAuthProvider

Represents the configuration file based inbound Basic Auth provider, which is an implementation of the auth:InboundAuthProvider interface.

  auth:InboundBasicAuthProvider inboundBasicAuthProvider = new;

A user is denoted by a section in the Ballerina configuration file. The password and the scopes assigned to the user are denoted as keys under the relevant user section as shown below.

 [b7a.users.<username>]
 password="<password>"
 scopes="<comma_separated_scopes>"
